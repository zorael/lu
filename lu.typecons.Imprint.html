Imprint

lu typecons

Mixin that copies all normal value types of a passed aggregate type into the mixin site scope.

Default values are carried over with structs, but not with classes and interfaces (as T.init is a valid value for structs but null in the case of classes).

Deliberately does not enforce a mixin constraint, so as to be useful if someone ever wants to copy an aggregate's fields into a function.

mixin template Imprint (
T
) if (
isAggregateType!T
) {}

Parameters

T

Type to copy members of.

Examples

1 struct Base
2 {
3     enum E { A, B, C }
4 
5     E enum_;
6     string s = "hello";
7     int i = 42;
8     float f = 3.14f;  // this value is lost
9     bool b = true;
10     Base* copy;
11 }
12 
13 struct Derived
14 {
15     mixin Imprint!Base;
16     bool b = false;  // overrides
17 }
static assert(__traits(hasMember, Derived, "enum_"));
static assert(__traits(hasMember, Derived, "s"));
static assert(__traits(hasMember, Derived, "i"));
static assert(__traits(hasMember, Derived, "f"));
static assert(__traits(hasMember, Derived, "copy"));
static assert(Derived.init.b == false);
